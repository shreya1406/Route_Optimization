1) Extract all the files to 1 folder
2) Open terminal and got to the folder in which all the files have been extracted
3) Start the python server by using the commnd : python -m SimpleHTTPServer. This will start your python server on the port 8000 by default. Suppose a different port is required(suppose 5000) the command can be: python -m SimpleHTTPServer 5000
4) Open Chrome and in the search bar type "http://localhost:<port_number>/main.html"
5) Click on "Get Data" and then on "Generate Routes"
